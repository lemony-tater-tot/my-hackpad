| **EC11 rotary encoders (20mm D-shaft)** | 1 | Rotary input with integrated push-switch |

| **Blank DSA keycaps** | 3 | 1u profile keycaps for the mechanical switches |
| **Custom Macro Pad PCB** | 1 | Custom KiCad routed circuit board |
| **Switch Plate & Enclosure** | 1 | 3d printed top-plate / 3D printed housing |
